# File Organizer Automation

A Python automation script that organizes files into folders based on their file extensions.

## About the Project

This project helps users organize files automatically.

The program analyzes the files inside a selected folder and moves them into category folders such as Images, Documents, Videos, Spreadsheets, Presentations, Audio, Compressed, Python, and Others.

The goal of this project is to practice Python automation and file system manipulation.

## Features

- Organize files based on their extensions
- Preview the organization before moving files
- Create category folders automatically
- Avoid overwriting files with the same name
- Generate a log file with the organization history
- Support multiple file types
- Allow the user to choose a folder to organize

## Technologies Used

- Python
- pathlib
- shutil
- File handling
- Automation logic

## How to Run

1. Clone this repository or download the project files.
2. Open the project folder in your terminal.
3. Run the command:

```bash
python main.py
```

4. Enter the path of the folder you want to organize, or press ENTER to organize the current folder.
